# Sense Skills

基于 SENSE 方法论的 Claude Code 技能包。

## 什么是 SENSE 方法

SENSE 是一套从问题澄清到行动决策的完整思维框架，由五个递进步骤组成：

| 步骤 | 名称 | 核心动作 |
|------|------|----------|
| **S** | Subject（问题） | 还原、分解、追问 |
| **E** | pErspective（视角） | 综合、解释、验证 |
| **N** | judgmeNt（判断） | 循环、分析、选择 |
| **S** | reverSe（逆转） | 阻碍、逆转、路径 |
| **E** | critiquE（批判） | 论证、良质、跃迁 |

## 技能列表

### `sense-human-loop`

SENSE 方法论引导的深度思考循环，分三个阶段：

1. **思考**：引导用户逐步完成 S-E-N-S-E 的思考过程
2. **学习**：遇到信息黑箱时，通过 subagent 或用户调查填充认知空白
3. **表达**：思考完成后，产出一份结构化的表达文档

**适用场景：** 复杂问题分析、重要决策、系统设计、项目规划

**触发方式：**
- 「帮我用 SENSE 分析这个问题：……」
- 「我想想清楚这件事……」
- 「sense loop」

---

### `sense-doc-writer`

基于 SENSE 方法论的结构化文档写作技能，将思考成果转化为清晰文档。

**支持文档类型：**
- 需求文档（PRD）
- 分析报告
- 决策备忘录
- 学习笔记

**触发方式：**
- 「把这些想法整理成一份需求文档」
- 「帮我写一份分析报告」
- 「sense 写作」

## 安装

```bash
# 安装所有技能
./install

# 查看可安装的技能列表
./install --list

# 卸载技能
./install --remove
```

安装脚本会在 `~/.claude/skills/` 目录下创建软链接，Claude Code 会自动识别并加载这些技能。

## 目录结构

```
sense/
├── skills/
│   ├── sense-human-loop/
│   │   └── SKILL.md        # 深度思考循环技能
│   └── sense-doc-writer/
│       └── SKILL.md        # 文档写作技能
├── install                  # 安装脚本
└── README.md
```

## 使用方式

安装完成后，在 Claude Code 对话中直接描述需求即可，Claude 会自动识别并使用对应的技能。

**示例对话：**

```
用户：我想用 SENSE 方法分析一下我们团队的效率问题。

Claude：好的，让我们开始 SENSE 思考循环。
        首先，让我理解一下你的问题……
```

```
用户：把刚才我们讨论的内容整理成一份需求文档。

Claude：好的，我来帮你整理。基于我们的讨论，
        核心问题是……我建议文档结构如下……
```
